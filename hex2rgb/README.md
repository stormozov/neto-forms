Конвертер цветов из HEX в RGB
===

Вам необходимо разработать конвертер цветов из HEX в RGB.

![Конвертер цветов](https://github.com/netology-code/ra16-homeworks/raw/ra-new/forms/hex2rgb/assets/preview.png)

## Интерфейс конвертера

При правильном вводе цвета он показывает его представление в формате RGB и меняет цвет фона на заданный:

![Цвет](https://github.com/netology-code/ra16-homeworks/raw/ra-new/forms/hex2rgb/assets/color.png)

Конвертер при вводе неправильного цвета в формате HEX должен сообщать об ошибке:

![Ошибка](https://github.com/netology-code/ra16-homeworks/raw/ra-new/forms/hex2rgb/assets/error.png)

Необходимо дожидаться ввода всех семи символов, включая решётку, чтобы принимать решение о том, показывать ошибку или менять цвет фона.

Стили и пример разметки вы можете найти в папке [markup](https://github.com/netology-code/ra16-homeworks/blob/ra-new/forms/hex2rgb/markup). Разметка дана для примера, вы можете реализовать её самостоятельно
